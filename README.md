# Система управления курьерской доставкой (DMA)


## Описание
Проектирование системы управления курьерской доставкой для стартапа, объединяющего заказы от магазинов и предприятий питания с курьерами для оперативной доставки клиентам. Система решает ключевые проблемы локдауна: резкий рост онлайн-продаж и нехватку быстрой доставки мелких партий товаров.

**Бизнес-цели:**
- Автоматизация сбора и распределения заказов между курьерами
- Сокращение времени доставки за счёт оптимизации маршрутов
- Прозрачный расчёт оплаты курьеров и взаиморасчётов с поставщиками
- Удобное мобильное приложение для курьеров с отслеживанием статусов заказов

## Что сделано
- Анализ 6+ стейкхолдеров (курьеры, диспетчеры, администраторы, поставщики, бухгалтерия)
- Выявление бизнес-требований и проблем «as is» → «to be»
- 8+ пользовательских историй с критериями приёмки для ролей курьера и диспетчера
- Луковичная диаграмма стейкхолдеров и контекстная диаграмма взаимодействия
- Диаграммы потоков данных (DFD level 0) и плавательные дорожки (swimlane)
- Диаграмма состояний объекта «Заказ» (создан → забронирован → в пути → доставлен)
- Выявление 8+ сущностей предметной области (Заказ, Курьер, Точка выдачи, Поставщик и др.)
- Проверка сущностей по матрице CRUD
- ER-диаграмма логической модели данных
- BPMN-диаграммы ключевых бизнес-процессов:
  - Приём и распределение заказов
  - Выполнение доставки курьером
  - Расчёт оплаты и передача данных в бухгалтерию
